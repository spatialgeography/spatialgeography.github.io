# Spatial Geography Lab — Website

Professional academic website for the **Spatial Geography Lab** built with [Quarto](https://quarto.org/).

## Research Focus

- **Deep Learning & Machine Learning** for remote sensing applications
- **Google Earth Engine** cloud-based geospatial analysis
- **QGIS** spatial analysis and cartographic production
- **Python & R** geospatial programming and data science
- **Field Survey & Ground Validation** for evidence-based outcomes

## Technology Stack

- **Framework:** [Quarto](https://quarto.org/) v1.8+
- **Theme:** Custom SCSS with Inter + Playfair Display typography
- **Icons:** Font Awesome 6 + Academicons
- **Deployment:** GitHub Pages (via `docs/` output directory)

## Local Development

```bash
# Preview locally
quarto preview

# Render the full site
quarto render
```

## Deployment

The site is configured to output to the `docs/` directory for GitHub Pages deployment. After rendering, commit and push to deploy.
