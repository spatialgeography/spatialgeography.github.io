## Key Scholars — Economic Geography

### Contributors and Their Contributions (NET Notes — Pulakesh Pradhan)

| Scholar | Contribution |
|:---|:---|
| **Derwent Whittlesey (1936)** | Agricultural classification — 13 types |
| **J.C. Weaver (1954)** | Statistical method of crop combination |
| **Doi (1959)** | Modified Weaver's method (sum of square deviation) |
| **Peter Scott (1963)** | Animals included in crop combination |
| **J.T. Coppock (1964)** | Animal-unit conversion in crop combination |
| **Kendall (1939)** | Yield per hectare as productivity measure |
| **Bhatia (1967)** | Yield index formula: Iya = Yc/Yr × 100 |
| **Enyedi & Shafi (1964 / 1972–74)** | Productivity index for 12 crops in India |
| **Jasbir Singh (1972–76)** | Multiple productivity techniques for India |
| **Alfred Weber (1909)** | Industrial Location Theory — minimum transport cost |
| **August Lösch (1954)** | Maximum profit; hexagonal market areas |
| **Von Thünen (1826)** | Agricultural Land Use Model — Isolated State |
| **David Ricardo** | Comparative Cost Advantage |
| **Lowry (1964)** | First transport and land use model |
| **Randhawa (1958)** | Agro-climatic region attempts |
| **Planning Commission India (1989)** | 15 Agro-Climatic Regions of India |
