### Data Presentation
* **Choropleth Maps**: Representing area-based statistical data with color gradients.
* **Isopleth Maps**: Continuous surfaces using isolines (isotherms, isohyets).
* **Proportional Symbol Maps**: Circle size proportional to variable magnitude.
* **Histograms, Box Plots & Scatter Plots**: Standard statistical graphics adapted for geographic data.
* **Triangular Diagrams**: Three-variable compositional data representation.
:::
