Overview

We are committed to open science and reproducible research. Below you’ll find curated resources including our open-source tools, Python/R code, GEE scripts, QGIS models, field survey protocols, and recommended learning materials.

Open-Source Tools & Code

LULC-DL Toolkit

Python library for deep learning-based land cover classification from satellite imagery, with built-in accuracy assessment and field validation utilities.

  • GitHub Repository
  • Documentation
  • pip install lulc-dl

GEE Workflow Library

Collection of Google Earth Engine JavaScript and Python scripts for change detection, composite generation, spectral indices, and time-series analysis.

  • GitHub Repository
  • Script Catalog
  • GEE Code Editor

R-Spatial Analysis Pack

R package with functions for spatial prediction, geostatistical analysis, accuracy assessment, and automated report generation for geographic research.

  • GitHub Repository
  • Documentation
  • devtools::install_github()

Field Survey Resources

Ground-Truth Collection Protocol

Our standardized field survey methodology for collecting GPS-referenced training and validation samples for remote sensing accuracy assessment.

  • Stratified random sampling design templates
  • GPS field data collection forms
  • Photo documentation guidelines
  • Accuracy assessment spreadsheet templates (OA, Kappa, F1)
  • Download Protocol PDF

QGIS Model Library

Collection of QGIS Processing models for terrain analysis, suitability mapping, MCDA, and cartographic layouts.

  • DEM derivative computation (slope, aspect, TWI, TPI, TRI)
  • Multi-Criteria Decision Analysis (AHP/TOPSIS) models
  • Network accessibility analysis models
  • Map layout templates for academic publications
  • Download Models

  • Jupyter Notebooks

Google Earth Engine Masterclass

Step-by-step guide from basic GEE operations to complex workflows: composites, indices, change detection, Random Forest classification, and time-series analysis.

  • Video Series (8 parts)

Advanced QGIS tutorials: spatial analysis, terrain modeling, MCDA, cartographic design, PyQGIS scripting, and Processing model development.

  • Video Series (10 parts)

From CNN basics to advanced architectures: image classification, semantic segmentation, object detection, and transfer learning with satellite imagery in Python.

Recommended Software Stack

GIS & Mapping

  • QGIS — Primary open-source GIS for visualization, analysis, and cartographic production
  • GRASS GIS — Advanced raster and vector processing, terrain analysis
  • Google Earth Engine — Cloud platform for planetary-scale analysis

Programming Languages

  • Python — GeoPandas, Rasterio, scikit-learn, TensorFlow, PyTorch, xarray
  • R — sf, terra, stars, tmap, caret, randomForest, rgee, tidyverse
  • JavaScript — GEE Code Editor, Leaflet, D3.js for web mapping

Remote Sensing

  • SNAP/ESA — Sentinel-1 SAR and Sentinel-2 optical processing
  • ERDAS IMAGINE — Advanced image classification and photogrammetry

Field Survey

  • GPS Essentials / SW Maps — Android GPS data collection apps
  • KoBoToolbox — Open-source field data collection platform
  • ODK (Open Data Kit) — Mobile data collection for ground-truth surveys
