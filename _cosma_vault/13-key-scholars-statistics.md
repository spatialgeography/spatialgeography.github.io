|:---|:---|

| **William Gosset** | T-test / Student's t-test (1908) |
| **R.A. Fisher** | Z-test; F-test; ANOVA |
| **M.O. Lorenz** | Lorenz Curve |
| **Corrado Gini** | Gini Coefficient |
| **W. Leontief** | Input-output analysis |
| **E. Shevky** | Social Area Analysis |
